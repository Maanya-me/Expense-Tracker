# Expense Tracker using Python

## 1. Objective

The objective of this project is to develop a simple command-line based Expense Tracker using Python that allows users to record and manage their daily expenses. The project helps users understand basic data management using Python lists and dictionaries while practicing fundamental programming concepts such as loops, conditional statements, and user input handling.

---

## 2. Problem Statement

Managing daily expenses manually can be difficult and time-consuming. This project provides a simple solution where users can store their expenses digitally, view all recorded expenses, and calculate the total amount spent through an easy-to-use menu-driven interface.

---

## 3. Features

* Add new expenses with date, category, description, and amount.
* Display all recorded expenses.
* Calculate the total amount spent.
* Menu-driven interface for easy interaction.
* Exit the application safely.

---

## 4. Technologies Used

* Python 3
* Lists
* Dictionaries
* Loops
* Conditional Statements
* User Input Functions

---

## 5. Working of the Project

1. The program starts by displaying a menu.
2. The user selects one of the available options.
3. If "Add Expense" is selected, the program asks for:

   * Date
   * Category
   * Description
   * Amount
4. The expense details are stored as a dictionary inside a list.
5. Users can view all stored expenses at any time.
6. The program calculates the total spending by summing all expense amounts.
7. The program continues until the user chooses the Exit option.

---

## 6. Advantages

* Easy to use.
* Beginner-friendly Python project.
* Helps keep track of daily expenses.
* Demonstrates the use of Python data structures.
* Can be extended with additional features.

---

## 7. Limitations

* Data is stored only during program execution.
* Expenses are not saved permanently.
* No option to edit or delete expenses.
* No graphical user interface.

---


## 8. Learning Outcomes

Through this project, the following concepts were practiced:

* Python Programming
* Lists and Dictionaries
* Loops
* Conditional Statements
* User Input Handling
* Basic Data Management
* Modular Problem Solving

---

## 9. Conclusion

The Expense Tracker project successfully demonstrates the implementation of a simple expense management system using Python. It provides an efficient way to record daily expenses, display stored records, and calculate total expenditure. The project serves as an excellent beginner-level application for understanding Python programming concepts and can be further enhanced by adding permanent storage, graphical interfaces, and data visualization features.
